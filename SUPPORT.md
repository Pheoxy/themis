# Support

Use the issue tracker for reproducible bugs, policy false positives, and feature requests.

Before asking for help:

- Run `nix flake check`.
- Read `docs/cli.md`.
- Include the command you ran and the generated validation report.
- Include the target repository rules that Themis should have followed.

Themis is intentionally conservative. A hard blocker is not automatically a bug; it may be the correct outcome when evidence is missing.
